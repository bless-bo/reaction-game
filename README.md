# 🐍 贪吃蛇大作战

经典怀旧的贪吃蛇游戏，网页版！

## 🎮 游戏规则

- 使用方向键或 WASD 控制蛇的移动方向
- 吃红色的食物可以增加长度和分数
- 每吃 5 个食物速度会加快
- 撞到墙壁或自己的身体游戏结束
- 记录最高分（保存在本地）

## 🕹️ 操作方式

**键盘：**
- `↑` `↓` `←` `→` 或 `W` `A` `S` `D` 控制方向
- `Enter` 或 `空格` 开始/重新开始

**手机：**
- 点击屏幕下方的方向按钮
- 或在画布上滑动控制方向

## 🏆 计分规则

- 每吃一个食物：+10 分
- 每 50 分速度提升一级
- 最高分保存在浏览器本地

## 🚀 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏！

```bash
# 或者使用本地服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```
