# ⚡ 反应速度挑战

一个简单有趣的 Web 反应速度测试游戏！

## 🎮 游戏规则

- 屏幕上会随机出现红色目标
- 越快点击目标，分数越高！
- 游戏共 10 轮
- 2秒内不点击目标会扣分

## 🏆 计分规则

| 反应时间 | 得分 |
|---------|------|
| < 200ms | 100分 |
| 200-300ms | 80分 |
| 300-500ms | 60分 |
| 500-800ms | 40分 |
| > 800ms | 20分 |

## 🚀 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏！

```bash
# 或者使用本地服务器
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 📱 响应式设计

游戏支持移动端触控，可以在手机上体验！

## 🎨 视觉效果

- 霓虹渐变背景
- 点击涟漪效果
- 动态分数显示
- 流畅的动画过渡
